Time, Space, Complexity

Time complexity and efficient is a measure of the amount of time an algorithm takes to execute as a function of its input size. It is an essential factor in determining the performance and scalability of a program, as inefficient algorithms can lead to slow execution times and unresponsive applications.

Space Complexity and O(n) is a parallel concept to time complexity. If we need to create an array of size n, this will require O(n) space. If we create a two-dimensional array of size n*n, this will require O(n2) space. In recursive calls stack space also counts.
